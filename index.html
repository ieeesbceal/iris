About IRIS 2025

IRIS 2025 is a prestigious 3-day event organized by the IEEE Signal Processing Society Student Branch Chapter of College of Engineering Attingal.

This hybrid-mode event features workshops, technical talks, competitions, and networking opportunities focused on signal processing, AI, and emerging technologies.

Join us for an immersive experience featuring keynote addresses by distinguished guests including Director of ICFOSS - Dr Sunil TT, sessions with industry experts, and hands-on workshops designed to enhance your technical skills.

Workshops

Gain hands-on experience in cutting-edge signal processing technologies

Eco Signals

Environmental monitoring using signal processing techniques to collect, analyze, and interpret environmental data.

Key Learning Outcomes:

  • Sensor integration for environmental parameters
  • Signal processing for environmental data analysis
  • Machine learning for environmental prediction

Resource Person: Aiswarya Lakshmi K.K., PhD Research Scholar, IIT Palakkad

AI-Driven Signal Processing

Combine AI with signal processing for intelligent applications using machine learning and deep learning techniques.

Key Learning Outcomes:

  • Apply ML algorithms to classify real-world signals
  • Handle time-series data effectively
  • Deploy AI models on edge devices

Resource Person: Dhanushkumar R, Microsoft Student Ambassador

Into the World of MYOSA

Explore MYOSA — an open-source framework for real-time signal acquisition, processing, and visualization designed for rapid prototyping and deployment in signal-processing applications.

Key Learning Outcomes:

  • Understand the MYOSA architecture and core components
  • Set up data acquisition pipelines and sensor integrations
  • Integrate signal-processing and ML modules for real-time analytics

Resource Person: Members from IEEE Sensors Council Kerala Chapter

Edge Computing & IoT

Focus on real-time signal processing at the edge using IoT technologies for data acquisition and analysis.

Key Learning Outcomes:

  • Interface sensors with microcontrollers
  • Apply signal processing on edge devices
  • Build IoT prototypes with real-time processing

Resource Person: Abhijith Benny, Hardware Engineer at Digital Core Technologies

Biomedical Signal Processing

Learn to process and analyze biomedical signals using Python and MATLAB for healthcare applications.

Key Learning Outcomes:

  • Understand ECG, EEG, and EMG signal characteristics
  • Apply filtering and denoising techniques
  • Extract features for diagnostic purposes

Resource Person: Aswin Sekhar CS, Research Officer at KK Women's and Children's Hospital

Cybersecurity

Learn essential cybersecurity skills covering network security, ethical hacking, cryptography, and threat detection.

Key Learning Outcomes:

  • Understand common attack vectors and vulnerabilities
  • Implement secure network protocols
  • Apply encryption techniques for data protection

Resource Person: Akhil Raj, Cybersecurity Analyst at TCS
